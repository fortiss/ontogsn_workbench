# Assets

This directory contains all static assets used by the OntoGSN Playground frontend application.

## Contents:
- `css/`: Cascading Style Sheets (CSS) for styling the application's user interface and visualizations.
- `data/`: Data files including OWL ontologies in Turtle (.ttl) format, SPARQL queries (.sparql) used to interact with the triplestore, and supporting Markdown documents.
- `js/`: JavaScript modules that implement the application's core logic, UI interactions, and data visualizations.
- `images/`: Various image files used throughout the application, such as icons, logos, or explanatory diagrams.
